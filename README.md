# SudokuGame
# 🎮 Sudoku 4x4 Game (Java Console)

## Deskripsi
Sudoku 4x4 adalah permainan logika sederhana berbasis command prompt (console) yang dibuat menggunakan bahasa pemrograman Java dengan konsep Object-Oriented Programming (OOP).

Dalam permainan ini, pemain diberikan papan berukuran 4x4 yang sebagian kotaknya sudah terisi angka 1 sampai 4, sedangkan sisanya masih kosong (ditandai dengan 0). Pemain harus mengisi seluruh kotak kosong hingga papan terisi penuh dengan aturan yang benar.

---

##  Aturan Permainan
- Setiap baris tidak boleh memiliki angka yang sama (1–4)
- Setiap kolom tidak boleh memiliki angka yang sama (1–4)
- Setiap blok 2x2 tidak boleh memiliki angka yang sama
- Pemain hanya boleh mengisi kotak yang masih kosong
- Game selesai jika seluruh papan terisi dengan benar

---

## Konsep yang Digunakan
Proyek ini menggunakan konsep Object-Oriented Programming (OOP), yaitu:
- **Class** sebagai blueprint game
- **Object** sebagai instance permainan
- **Encapsulation** untuk menyimpan data board
- **Method** untuk logika permainan seperti validasi, input user, dan pengecekan kemenangan

---

##  Tujuan Game
Tujuan dari game ini adalah:
- Melatih kemampuan berpikir logis
- Meningkatkan pemahaman algoritma dasar
- Mengenalkan konsep pemrograman berbasis objek (OOP)
- Melatih problem solving

---

##  Teknologi
- Bahasa: Java
- Interface: Command Line Interface (CLI / Console)
